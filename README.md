Ruby API Client for the [SendCloud](https://www.sendcloud.nl) delivery platform.


## Installation

Add this line to your application's Gemfile:

    gem 'sendcloud-ruby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sendcloud-ruby
